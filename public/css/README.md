# CSS

Стили для приложения "Точка Опоры".

## Файлы
- `style.css` - глобальные стили, переменные, шрифты, модальные окна
- `dashboard.css` - стили главной панели с карточками ситуаций
- `create-edit.css` - стили редактора ситуаций и шагов помощи
- `session.css` - стили сеанса (прогресс-бар, дыхательные упражнения)
- `profile.css` - стили профиля, аватара, контактов
