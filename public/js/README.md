# JavaScript

Клиентская логика приложения "Точка Опоры".

## Файлы
- `storage.js` - работа с localStorage (пользователи, ситуации, контакты, прогресс)
- `script.js` - авторизация, регистрация, анимации
- `dashboard.js` - отображение карточек, фильтрация, запуск сеансов
- `create-edit.js` - создание/редактирование ситуаций и шагов
- `session.js` - логика сеанса (дыхание, заземление, прогресс)
- `profile.js` - профиль, аватар, контакты, экспорт/импорт
- `toast.js` - уведомления и диалоги подтверждения

## Зависимости
- Все модули используют глобальный объект `window.Storage`
- `toast.js` предоставляет `showToast()` и `showConfirm()`
