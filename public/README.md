# Public

Корневая директория фронтенд-части приложения "Точка Опоры". Содержит все статические файлы, HTML-страницы и клиентский код.

## Структура
- `css/` - стили компонентов
- `js/` - клиентские скрипты
- `images/` - иконки и графика
- `fonts/` - шрифты приложения
- `media/` - аудиофайлы (фоновые звуки)

## HTML файлы
- `index.html` - главная страница / вход
- `dashboard.html` - панель управления ситуациями
- `create-edit.html` - создание/редактирование ситуации
- `session.html` - сеанс помощи
- `profile.html` - профиль пользователя
- `manifest.webmanifest` - PWA манифест
