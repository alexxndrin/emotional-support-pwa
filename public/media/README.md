# Media

Аудиофайлы для фонового сопровождения во время сеансов.

## Файлы
- `rain.mp3` - звук дождя
- `forest.mp3` - лесные звуки
- `brown.mp3` - коричневый шум
- `white.mp3` - белый шум

## Требования
- Формат: MP3
- Размер файлов не более 2 МБ
- Пользователи могут загружать свои звуки

## Использование
Фоновые звуки привязываются к конкретным ситуациям и автоматически запускаются во время сеанса помощи
