# Точка Опоры

Автономный PWA-помощник при тревоге и панических состояниях.

## Описание
Приложение помогает пользователям справляться с паническими атаками и тревогой через пошаговые протоколы: заземление 5-4-3-2-1, дыхательные упражнения, фразы-якоря и действия в реальности.

## Особенности
- Полностью автономная работа (PWA, Service Worker)
- Все данные хранятся локально на устройстве
- Создание персонализированных ситуаций
- Экспорт/импорт резервных копий
- Фоновые звуки для сеансов
- Защита от случайного закрытия (fullscreen режим)
